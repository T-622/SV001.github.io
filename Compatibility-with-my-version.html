A question I get asked all the time is: What version of minecraft does this require; is it compatible with my platform?

Luckily, I support *almost* every platform that can run a version of minecraft!

Java Edition

Yes! We have full compatibility for Java edition! This means, any computer using a Java edition will be compatible. Use the "How to connect" button below to obtain connection details!

Buy Java (~35$ CAD) How to connect!

Don't have Java? Don't sweat it! We also support other versions too!

Bedrock Edition / PE

Yes! We also have full support for Bedrock too! We utilise the Geyser cross-platform plugin to do so. Use the "How to connect" button below to obtain the connection details!

Buy Bedrock (~27$ CAD) How to Connect!

This means most Switches, Mobile devices and consoles can now join!
